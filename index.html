Z - reset board (and to change board size)
Scoring The scoring is calculated using the formula:
Let board size be bs
Let unique cells visited be ucv
Let bonus points be bp (each bonus cell absorbed gives 1 point)
Each difficult has a difficulty multiplier (dm). Easy: 0.8, Medium: 1.0, Normal: 1.2, Hard: 1.4, Extreme: 1.6, Impossible: 1.8
Let turns survived be ts. The game stops when more than 1/4 of the board size turns have passed, so this stays constant relative to board size.
The formula is: ((2 * bs + ucv + (3 * bp)) * dm * ts).
